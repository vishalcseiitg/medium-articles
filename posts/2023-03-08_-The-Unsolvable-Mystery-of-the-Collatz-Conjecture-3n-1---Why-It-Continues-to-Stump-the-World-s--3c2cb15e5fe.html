Introduction

A. Definition of the Collatz Conjecture

The Collatz Conjecture, also known as the 3n+1 problem or the Syracuse Conjecture, is a mathematical conjecture that has puzzled mathematicians for decades. The conjecture posits that if you follow a simple set of rules, no matter what positive integer you start with, you will eventually reach the number 1. The rules are as follows: if the number is even, divide it by 2, and if the number is odd, multiply it by 3 and add 1. Repeat this process with the new number until you reach 1.

B. Brief history of the problem

The Collatz Conjecture was first proposed by German mathematician Lothar Collatz in 1937, but it was largely ignored until the 1950s. In the years since many mathematicians have attempted to prove or disprove the conjecture, but as of yet, no one has been successful. The conjecture has become a famous example of a simple problem that is deceptively difficult to solve, and it has captured the imaginations of mathematicians and non-mathematicians alike.

C. Importance of the problem

While the Collatz Conjecture may seem like a purely academic problem, it has important implications for other areas of mathematics. For example, it is related to the study of automata and has been used to construct examples of automata with unusual properties. Additionally, the Collatz Conjecture connects to number theory and the study of prime numbers. It has also generated interesting patterns in computer graphics and other visual arts. Finally, the problem is fascinating in its own right, and the fact that it has resisted solution for so long only adds to its allure.

Statement of the problem

A. Explanation of the rules

As mentioned earlier, the rules of the Collatz Conjecture are simple: start with any positive integer, and then repeatedly apply the following steps until you reach the number 1:

  • If the number is even, divide it by 2.
  • If the number is odd, multiply it by 3 and add 1. The conjecture posits that no matter what number you start with, you will eventually reach 1.
def collatz(n):
    """Function to implement the Collatz Conjecture"""
    print(n)
    while n != 1:
        if n % 2 == 0:
            n = n // 2
        else:

Despite the apparent simplicity of the rules, the Collatz Conjecture has proven extremely difficult to solve. While the conjecture has been tested extensively with computers, and there is strong evidence that it is true for all numbers up to a certain limit, no one has been able to prove that it holds for all numbers. Mathematicians have yet to prove that the process does not eventually reach a cycle of repeating numbers. The challenge of finding a pattern in the Collatz Conjecture has made it a favorite problem of mathematicians and amateurs alike.

Attempts to solve the problem

A. Computational searches

Computational searches have been used extensively to test the Collatz Conjecture for a vast range of numbers. These searches have not found any counterexamples, which provides strong evidence that the conjecture is true for all numbers up to a certain limit. In particular, it has been tested for all numbers up to 2⁶⁸, and no counterexamples have been found. This limit is so high that it would take more than 300 years for the fastest supercomputer to test all numbers up to it.

B. Theoretical approaches

Various theoretical approaches have been used to try to solve the Collatz Conjecture. One such approach is looking for a “structural” property of the numbers generated by the process, such as certain cycles or patterns. Another approach is to try to show that certain sequences of numbers cannot occur in the process. However, all of these approaches have so far been unsuccessful in solving the problem.

C. Why is the problem still unsolved

The Collatz Conjecture remains unsolved because of the sheer complexity of the problem. While the conjecture seems simple, it is incredibly difficult to determine whether there is any underlying structure or pattern to the numbers generated by the process. In addition, the search space for the conjecture is virtually infinite, and it is impossible to test all numbers. The problem's difficulty has made it one of the most famous and long-standing unsolved problems in mathematics. Despite the many attempts to solve it, the Collatz Conjecture remains an open problem today.

import matplotlib.pyplot as plt

def collatz(n):
    seq = [n]
    while n != 1:
        if n % 2 == 0:
            n = n // 2
        else:
            n = 3*n + 1
        seq.append(n)

num_primes = []
for n in range(1, 101):
    seq = collatz(n)
    primes = [x for x in seq if all(x % i != 0 for i in range(2, int(x**0.5)+1))]
    num_primes.append(len(primes))

def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num**0.5)+1):
        if num % i == 0:
            return False
    return True
